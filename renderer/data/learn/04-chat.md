# Chat with ACE

Chat is how you speak to ACE. Every message runs against the current state of your vault, so nothing has to be re-explained. Your files are the context.

**Slash commands are rituals.** Type `/` and the composer surfaces the full set — `/start` to ground the day, `/brief` for a morning read, `/eod` to close it out, and the rest. These aren't shortcuts. They're the practices the system runs on.

**Model, effort, permissions.** Above the composer you can dial which model you're talking to, how hard it should think, and what it's allowed to touch on your machine. The defaults are tuned for daily use — leave them alone until you have a reason to change them.

**Every message lives inside a session.** Sessions are containers with a beginning and an end, and they keep the conversation bounded so context stays clean. The next lesson covers how that works.

For now, the invitation is simple. Send a message. Or type `/brief` and let the system show you what today actually holds.

*The vault listens. Speak to it.*
